# Web Phishing Detection Tool

## Overview

This project is a simple Web Phishing Detection Tool developed using Python. It analyzes website URLs and identifies whether they appear safe or suspicious using basic phishing detection techniques.

## Features

- Website URL Analysis
- Safe / Suspicious Detection
- Risk Analysis
- HTTP Status Check
- Simple Python Implementation

## Technologies Used

- Python
- Requests Library

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
py phishing_detector.py
```
